### Installation:

- **To clone the project**

```
git@github.com:djmaluy/milestep-test-frontend.git

```

- **Go to the folder**

```
cd milestep-test-frontend  and execute the command:

npm install

```

### Run the application:

- **For quick application run use this command**

```
npm start

```
